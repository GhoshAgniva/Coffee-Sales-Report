# ☕ Coffee Shop Sales Dashboard (Microsoft Excel)

![Dashboard](Sales%20Dashboard.png)

## 📌 Project Overview

This project is an interactive **Coffee Shop Sales Dashboard** built entirely in **Microsoft Excel**.

The dataset was first cleaned and transformed using **Power Query**, followed by data analysis using **Pivot Tables**, **Pivot Charts**, **Slicers**, and interactive dashboard components.

The primary objective of this project is to analyze retail coffee shop sales data and generate actionable insights that help understand customer purchasing behavior, store performance, product popularity, and sales trends.

# 📊 Dashboard Preview

The dashboard allows users to interactively filter data using **Month** and **Day** slicers.

Users can instantly explore sales performance across different dimensions without writing any formulas or SQL queries.

---

# 📈 Dashboard Features

The dashboard includes the following analyses:

### 📅 Sales Trend Analysis
- Quantity sold by hour
- Identify peak business hours
- Analyze customer purchase patterns throughout the day

---

### 🍩 Product Category Distribution

Analyze how different product categories contribute to total sales.

Examples:
- Coffee
- Tea
- Bakery
- Drinking Chocolate
- Flavours
- Packaged Chocolate

---

### 🏆 Top 5 Best-Selling Products

Displays the highest revenue-generating products.

Useful for:
- Inventory planning
- Product promotions
- Menu optimization

---

### 📍 Store Performance

Compare sales across different store locations:

- Astoria
- Hell's Kitchen
- Lower Manhattan

This helps identify the highest-performing branch.

---

### 👣 Store Footfall Analysis

Displays customer footfall across store locations.

Useful for understanding customer traffic and branch popularity.

---

### 📦 Product Size Distribution

Visualizes customer preference for different drink sizes:

- Small
- Regular
- Large
- Not Defined

---

### 📆 Daily Sales Analysis

Identify which day of the week generates the highest sales.

---

# 🎯 Business Questions Answered

This dashboard answers the following business questions:

- How does sales quantity vary by hour of the day?
- Which day generates the highest sales?
- Which product categories contribute the most revenue?
- Which are the top 5 best-selling products?
- Which store location performs the best?
- Which branch receives the highest customer footfall?
- What product sizes do customers prefer?

# 🧹 Data Cleaning Using Power Query

The raw dataset was cleaned using Power Query by performing:

- Removing duplicate records
- Handling missing values
- Correcting data types
- Formatting date and time columns
- Creating Month and Day fields
- Preparing clean data for Pivot Tables
